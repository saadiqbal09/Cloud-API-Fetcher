Cloud-API-Fetcher

A Python-based project to fetch device data from a cloud API and publish
processed parameters to an MQTT broker.

Features
- Fetch data from cloud REST API
- Parse JSON response
- Map parameters using CSV configuration
- Publish data to MQTT broker
- Auto-reload config on CSV change

Tech Stack
- Python
- REST API
- MQTT
- CSV Configuration
